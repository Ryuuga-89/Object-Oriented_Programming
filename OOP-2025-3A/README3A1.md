# 問題3A-1

`Main3A1`クラスの`main`メソッドは実行時引数を順に一行ずつ表示している。
これを、引数ごとにコンマ区切りで文字列を分け、分けた文字列を一行ずつ表示するようにせよ。
ただし、表示する各文字列は先頭と末端にスペースが入らないように除去すること（間のスペースは除去しない）。

### 実行例
コマンドライン上でスペースを挟むと別の引数扱いになるため、以下の実行例では必要に応じてダブルクオートで囲っている。


    > java Main3A1 ABC DEF
    ABC
    DEF
    > java Main3A1 "ABC, DEF"
    ABC
    DEF
    > java Main3A1 "  AB,CD    , EF G "
    AB
    CD
    EF G
